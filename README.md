# Calrma Calendar

Moderan kalendar sa glassmorphism dizajnom i full funkcionalnostima.

## Features

- Dodavanje/brisanje događaja
- 4 kategorije sa bojama (Posao, Lično, Škola, Važno)
- Dark/Light mode
- Pretraga događaja
- Responsive dizajn
- Local storage

## Tech Stack

React • Vite • CSS3

## Kako pokrenuti
```bash
npm install
npm run dev
```
